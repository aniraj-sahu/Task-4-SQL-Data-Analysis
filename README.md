# Task 4: SQL for Data Analysis

## 📌 Objective

The objective of this project is to analyze an ecommerce dataset using SQL by writing queries to extract meaningful business insights. The project demonstrates data exploration, sales analysis, profitability analysis, inventory analysis, customer satisfaction analysis, and advanced SQL concepts.

---

## 📂 Dataset

**Dataset:** Ecommerce Dataset

The dataset contains product, brand, pricing, inventory, customer ratings, promotions, and monthly sales information.

---

## 🛠️ Tools Used

- MySQL Workbench
- MySQL Server

---

## 📊 SQL Concepts Covered

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions
- Subqueries
- Views
- Indexes
- INNER JOIN
